# KIRA
Overview

KIRA is a mobile app that helps international high school students in the U.S. plan their academic and career paths through clear, personalized roadmaps. It simplifies college and career planning by organizing goals, opportunities, and requirements in one place.

Problem

Students often struggle to connect high school decisions to future careers because information is scattered and hard to navigate.
Solution
KIRA provides structured, personalized guidance that helps students explore careers, plan academics, and track progress directly from their phone.

Key Features

Career path roadmaps

College planning tools

Scholarships and opportunities hub

Student progress tracker

Resume support

Tech Stack

React Native (Expo)

Firebase

Firestore

GitHub

Goal

To make career and college planning simple, structured, and accessible for international students through a mobile-first platform.
